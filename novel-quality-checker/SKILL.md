# Novel Quality Checker — 章节质量审核

## 33 维度审核

运行 `scripts/quality_check.py` 对指定章节或全书进行审核。

### 快速检查（5项核心）
```bash
python3 scripts/quality_check.py --file <章节文件> --quick
```
- 字数达标（≥12000）
- 中文纯度（>95%汉字）
- AI标记检测
- 模板化结尾检测
- 角色名一致性

### 全量审核（33维度）
```bash
python3 scripts/quality_check.py --file <章节文件> --full
# 或批量
python3 scripts/quality_check.py --dir <目录> --full
```

### 审核维度

| 维度 | 检查方法 | 标准 |
|------|---------|------|
| 字数 | Python计数 | ≥12000字 |
| 重复率 | n-gram相似度 | <10%段落重复 |
| AI标记词 | 关键词匹配 | 0个 |
| 模板化结尾 | 正则匹配 | 0个 |
| 角色名 | 全名搜索 | 全文统一 |
| 纯中文 | 字符分类 | >95%汉字 |
| 段落长度 | 段落统计 | 无超长段落 |
| 对话比例 | 引号内容统计 | 30-50% |
| 动作描写密度 | 动词统计 | 合理分布 |
| 环境描写密度 | 场景词统计 | 适当点缀 |
| 情节推进 | 事件密度 | 有进展 |
| 金手指提及 | 关键词搜索 | 出现≥2次 |
| 冲突/张力 | 冲突词统计 | 有矛盾推动 |
| 节奏变化 | 段落长度方差 | 有张有弛 |
| 视角一致性 | 代词检查 | 无视角跳 |

### 输出格式

```
╔══════════════════════════════╗
║  第X章 YYY 质量审核报告      ║
╚══════════════════════════════╝

✅ 通过: 28/33
⚠️  警告: 3/33  
❌ 不合格: 2/33

[详细列表...]

总结: 总体质量{优/良/中/差}
```

## Scripts
- `scripts/quality_check.py` — 完整审核脚本

## 审核流程

1. 运行脚本 → 生成报告
2. 阅读报告 → 标出不合格项
3. 针对不合格项修复（退回给生成环节或手动修改）
