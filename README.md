## 🏗️ Pipeline

```
Think → Plan → Build → Review → Ship → Retro
```

- **Think**: Question framing, emotional arc placement
- **Plan**: Extract outline, allocate word counts, constraints
- **Build**: Main generation via free APIs
- **Review**: 3-stage — Auditor → Proofreader → Quality Director
- **Ship**: Chapter with quality report
- **Retro**: Post-chapter review, update lessons
